# Documents ADK Kasting

Ce dossier contient les documents téléchargeables pour le formulaire d'inscription.

## Autorisation parentale

Le fichier `autorisation-parentale-ADK.pdf` doit être créé avec le contenu suivant :

### Contenu du document

```
ADK KASTING
Autorisation parentale pour mineur

Je soussigné(e) : ________________________________________________
                    (Nom et prénom du parent/tuteur légal)

Agissant en qualité de : ☐ Père  ☐ Mère  ☐ Tuteur légal

Adresse : __________________________________________________________

Code postal : ____________  Ville : _______________________________

Téléphone : __________________  Email : ___________________________


AUTORISE mon enfant :

Nom : _____________________________________________________________

Prénom : __________________________________________________________

Né(e) le : ___ / ___ / _________


À s'inscrire sur la plateforme ADK Kasting et à participer à :

☐ Castings pour films de long métrage
☐ Castings pour films de court métrage
☐ Castings pour publicités
☐ Castings pour séries télévisées
☐ Séances photos professionnelles
☐ Autres projets audiovisuels


Je déclare avoir pris connaissance :
- Des conditions générales d'utilisation de la plateforme ADK Kasting
- Du traitement des données personnelles de mon enfant
- Des modalités de participation aux castings


Je m'engage à :
- Accompagner mon enfant lors des castings et tournages
- Informer ADK Kasting de tout changement de situation
- Respecter les engagements pris au nom de mon enfant


Fait à : ________________________  Le : ___ / ___ / _________


Signature du parent/tuteur légal :
(Précédée de la mention "Lu et approuvé")




---
Document à retourner signé lors de l'inscription
ADK Kasting - www.adk-kasting.com
```

## Instructions

Pour créer le PDF :
1. Utiliser un logiciel comme LibreOffice, Microsoft Word, ou Canva
2. Mise en page A4, marges standard
3. Police professionnelle (Arial, Helvetica, Inter)
4. Logo ADK en en-tête si disponible
5. Exporter en PDF
6. Nommer le fichier : `autorisation-parentale-ADK.pdf`
